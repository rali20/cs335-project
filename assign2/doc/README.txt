To run program do following :
  $ cd <project-top>
  $ make
  $ ./src/myparser.py --input ../tests/test1.go --out test1.ps

Options of ./src/myparser.py :
  --help
    prints this same message
  
  --input <file.go>
    give input go souce file    
  
  --out <file.ps>
    give name of the output postscript file
